# velogears
[![Build Status](https://travis-ci.org/VladimirTaytor/velogears.svg?branch=develop)](https://travis-ci.org/VladimirTaytor/velogears)
 
 Open source online store build with Node.js and Express.
 
 ### Technologies:
  1. **Backend**:
        + Node.js; 
        + Express.js;
        + Sequelize;
        + MySQL;
  2. **Frontend**:
        + Jade;
        + Bootstrap;
        + LESS;
  3. **Testing**:
        + Mocha;
        + Expect.js;
